# 🏥 Healthcare Patient Risk Prediction

## 🎯 Problem Statement
Predict whether a patient is likely to develop diabetes based on medical attributes.

---

## 📊 Dataset
- Pima Indians Diabetes Dataset
- 768 patient records with medical features

---

## 🧠 Approach
- Data Cleaning (handled missing values)
- Exploratory Data Analysis (EDA)
- Feature correlation analysis
- Model building (Logistic Regression & Random Forest)
- Model evaluation

---

## 📈 Key Insights
- Glucose is the most influential feature
- Higher BMI and age increase risk
- Dataset is imbalanced

---

## 🤖 Model Results
- Logistic Regression Accuracy: 75%
- Random Forest Accuracy: 76%
- Random Forest performs better in detecting diabetic patients

---

## 💡 Conclusion
Random Forest is more suitable for healthcare prediction due to better recall and ability to capture complex relationships.
